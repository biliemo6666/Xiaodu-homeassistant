# Xiaodu-homeassistant
此项目有apgmer用户的hass-xiaodu项目二改[apgmer](https://github.com/apgmer/hass-xiaodu)

## 修改内容
支持了对风扇及自定义设备的支持。

## 使用方法
1.安装Xiaodu改版，首先打开Homeassistant，然后打开Hacs，右上角三点，选择自定义仓库(Custom repositories)，里面仓库填写"https://github.com/biliemo6666/Xiaodu-homeassistant"，类型(Category)选择集成(Integration)，点击新增，等待一会后会显示一个新的仓(小度 Home Assistant 集成)，关闭窗口，在列表里找到(小度 Home Assistant 集成)，点击后点击下载按钮，在窗口里选择下载，等待下载完毕重启Homeassistant。

2.重启好后在设置->设备与服务->右下角添加集成，在搜索框里搜索(小度)

2.在[小度](https://xiaodu.baidu.com/saiya/smarthome/index.html)网站里登录自己的小度账号，登录好后按键盘上的F12或者浏览器右上角三点->更多工具->开发者工具 打开开发者工具
